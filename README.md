# 🚀 阿里云盘多账号自动签到 (青龙版)

基于 Python 实现的阿里云盘每日自动签到脚本，支持多账号，支持 PushPlus 微信通知。

## 🌟 功能特点
- 支持环境变量读取 Token，安全不泄露。
- 支持多账号同时签到（换行或 & 分隔）。
- 集成 PushPlus 通知模块，签到状态直达微信。
- 自动识别青龙 Env 环境，一键添加任务。

## ⚙️ 环境变量配置

| 变量名              | 描述                   | 示例                      |
| :------------------ | :--------------------- | :------------------------ |
| `ALI_REFRESH_TOKEN` | 阿里云盘 refresh_token | `32位Token1` (多账号换行) |
| `PUSH_PLUS_TOKEN`   | PushPlus 的用户令牌    | `...`                     |

## 🛠️ 青龙面板部署

1. **拉取仓库** (请替换为你的仓库地址):
